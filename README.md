## 시작 가이드 (동료용)

### 1) 사전 준비
- Node.js `20+` 권장
- npm 사용 (프로젝트 기본 패키지 매니저)

### 2) 프로젝트 받기
```bash
git clone <레포지토리_URL>


3) 의존성 설치
npm install

4) 로컬 개발 서버 실행
npm run dev
실행 후 터미널에 출력되는 로컬 주소(예: http://localhost:5173)로 접속

5) 빌드 / 린트 확인
npm run build
npm run lint


린트 미리 설정했습니당! 일단 널널한 규칙으로 했..
커밋 안되면 린트오류가 있는 것이니 해결하고 푸쉬해주세욥
